# minecraft-bots

An assortment of bots for use in Minecraft

## Installation

Run npm install

## Deploying

To deploy simply push to master

## Usage

To run a bot in the server, start it in a "screen" using `nodemon script.js`

For example:

```
screen -S bot-bob
cd minecraft-bots
nodemon bot-bob.js
ctrl a+d
```
